Programando o Ano Automático no Rodapé do Site

  Existem muitos códigos abertos de sites estáticos na web. Porém, como sou teimoso, resolvi desenvolver mal feito o meu. Depois de dobrar o número de cabelos brancos na minha cabeça com a desorganização de diretórios, conforme registrado nos meus commits, me vi com um desejo: quero o ano no rodapé do site.

  Recém virado o ano, 2025 bem bonito lá embaixo, e quando chegar 2026? As minhas páginas são todas "cópias" do index com interior modificado para dar impressão de navegação no mesmo ambiente. Porém, hoje com meia dúzia de páginas é só ir uma a uma e editar o ano para 2026. Entretanto, conforme cresce o número de artigos, serão mais páginas para eu editar. Até que cheguei na técnica em JavaScript que utiliza a captura da data atual e retorna o ano extraído dela. Optei por utilizar o window.onload para chamar o script apenas no carregamento da página. Agora é só adicionar o seguinte código ao rodapé de cada página nova e está resolvido o problema:

HTML

<span id="ano"></span>

JS

window.onload = function() { document.getElementById("ano").innerHTML = new Date().getFullYear(); }
