Frase invertida

  Você deve pegar uma frase como "Dog bites man" e invertê-la alterando a ordem das palavras, resultando em "man bites Dog".

Resolução 
                        
public class WordReversal {

    public static void main(String[] args) {
        String frase = "Dog bites man";
        String[] palavras = frase.split(" ");

    System.out.println(palavras[2] + " " + palavras[1] + " " + palavras[0]);
    }
}

  Criada a classe WordReversal, já declarando no método principal uma String 'frase' a ser analisada e um Array palavras recebendo frase.split(" ") como argumento um espaço em branco como limite de corte da String frase, formando assim um Array com cada palavra sendo endereçada de 0 a 2. Bastando agora fazer uma saída inversa de 2 até 0.

Saída:

man bites Dog

Soma de 5 valores inseridos pelo usuário

  Você deve permitir que o usuário digite 5 valores. A saída do programa deve retornar a soma dos 5 valores informados.

Resolução 
                            
import java.util.Arrays;
import java.util.Scanner;

public class ArrayUnidimensional {

    public static void main(String[] args) {
        int [] num = new int[5];

        System.out.println("---Digitando 5 valores---");
        
        Scanner valores = new Scanner(System.in);
        
        for (int c = 0; c < num.length; c++) {			
            System.out.print("Digite o " + (c + 1) + "º valor: ");				
            num[c] = valores.nextInt();
        }
        
        valores.close();
        
        int soma = 0;
        for (int i = 0; i < num.length; i++) {			
            soma += num[i];
        }
                
        System.out.println("A soma de " + Arrays.toString(num) + " é igual a " + soma + ".");
    }
}

  Criado um array de 5 posições, Abrindo um Scanner para armazenar os dados inseridos pelo usuário. Optei por iniciar um laço de repetição que percorrerá todas as posições do array de 0 a 4, iterando cada valor inserido. Após a coleta, vamos ao tratamento dos dados, iniciando uma variável com valor 0 e dando inicio a outro laço de repetição agora realizando a soma iterativa novamente percorrendo todos os endereços do array de 0 a 4 e finalmente imprimindo a resposta ao usuário, utilizando também o método Arrays.toString() para otimizar a saída mostrando os valores digitados.

Saída:

---Digitando 5 valores---
Digite o 1º valor: 1
Digite o 2º valor: 2
Digite o 3º valor: 3
Digite o 4º valor: 4
Digite o 5º valor: 5
A soma de [1, 2, 3, 4, 5] é igual a 15.

Criando uma matriz 3x3 com dados do usuário e somando os valores

  Você deve pedir 9 múmeros para o usuário, em uma ordem de 3 linhas para 3 colunas, montando uma matriz 3x3, exibi-la em formato tabular e somar seus valores.

Resolução 
import java.util.Scanner;

public class ArrayBidimensional {

    public static void main(String[] args) {
        int [][] matriz = new int [3][3];
        
        System.out.println("----Inserindo 3 valores em 3 linhas----");
        
            Scanner valores = new Scanner(System.in);
        
        for (int i = 0 ; i < matriz.length; i++) {
            System.out.println("Linha " + (i + 1));
            
            for (int j = 0; j < matriz[i].length; j++) {
                System.out.print("Valor " + (j + 1) + ": ");
                
                matriz[i][j] = valores.nextInt();
            }			
        }
        
        valores.close();
        
        System.out.println("---Matriz no formato tabular---");
                
        for (int c = 0; c < matriz.length; c++) {
            for (int d = 0; d < matriz[c].length; d++) {
                System.out.print(matriz[c][d] + " ");
            }
            System.out.println();			
        }
        
        int soma = 0;
            for (int e = 0; e < matriz.length; e++) {
                for (int f = 0; f < matriz[e].length; f++) {
                    soma += matriz[e][f];
                }
            }
        System.out.println("Soma dos elementos da matriz = " + soma);

    }
}

Para solucionar esse desafio eu dividi em três partes:

  • Parte 1 - Criar a matriz 3x3 e fazer a coleta dos dados com Scanner. Utilizei a técnica de fazer dois laços de repetição indentados, para percorrer as linhas no primeiro e o segundo as colunas, armazenando os valores nas posições de 0 a 2 dos arrays de ambas dimensões.

  • Parte 2 - Fazer a montagem da matriz no formato tabular 3x3, exatamente com a mesma técnica de coleta de informações, porém agora distribuindo os valores e adicionando uma quebra de linha ao final de cada saída do segundo laço para induzir a impressão do formato tabular.

  • Parte 3 - Realizar a soma dos elementos, iniciando com uma declaração de variável valendo 0 e, mais uma vez, aplicando a técnica de percorrer todas as linhas e colunas com dois laços de repetição indentados. dessa vez ao satisfazer cada condição do segundo laço, realizar uma iteração dos valores armazenados nos endereços. E ao final imprimindo a soma.

Encontre a palavra pela posição

  Você deve permitir que o usuário digite uma frase e em seguida peça qual a palavra ele quer que o programa retorne através da posição dela. Exemplo: 'quero café'. Posição 2 retornará 'café'.

Resolução 
import java.util.Scanner;

public class FindWord {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Digite uma frase: ");		
        String entrada = sc.nextLine();		
        String[] fracionado = entrada.split(" ");		
        
        System.out.print("Mostrar a palavra nº: ");				
        int numero = sc.nextInt() - 1;	
        
        sc.close();
        
        if(numero >= 0 && numero < fracionado.length) {
            System.out.println("A palavra nº " + (numero + 1) + " é: " + fracionado[numero]);
        } else {
            System.out.println("A frase possui " + fracionado.length + " palavras! Digite um número entre 1 e " + fracionado.length);
        }		
    }
}

Para esta resolução, eu adotei quatro passos:

  • 1º Passo - Abrir um Scanner para solicitar entradas de dados pelo usuário.

  • 2 º Passo - Solicitando e armazenando uma frase ao usuário. Criando um Array, convertendo a String inserida pelo usuário com o método split, passado como parâmetro um espaço vazio, delimitando o corte de cada palavra pelo espaço da frase, tornando cada palavra rastreável pelo programa.

  • 3 º Passo - Solicitando ao usuário a inserção de qual numeração da posição da palavra que ele deseja que o programa retorne. Com um detalhe, a entrada que será um números partindo de 1 e a variável 'numero' já tratada para corresponder ao índice que se inicia com 0, portanto, recebendo o valor inserido - 1.

  • 4º Passo - Agora para a saída, fiz uma condição de caso o valor digitado seja menor ou igual a zero ou maior que a quantidade de palavras armazenadas no array, apresente uma mensagem de erro apresentando a quantidade de palavras existentes na frase.

Saída:

Digite uma frase: Digitei tudo errado
Mostrar a palavra nº: 2
A palavra nº 2 é: tudo

Saída com tratamento de erro:

Digite uma frase: Estudei Java o dia inteiro
Mostrar a palavra nº: 8
A frase possui 5 palavras! Digite um número entre 1 e 5

Buscador

  Você deve fazer um programa que busque o que foi digitado pelo usuário em um texto que você vai definir e retorne a quantidade de termos encontrados.

Resolução 
import java.util.Scanner;

public class WordSearch2 {

	public static void main(String[] args) {
		String frase = "Esse texto será utilizado para fazer a varredura de quantidades de texto iguais . Para tanto , o texto é esse";
		
		String [] fracionado = frase.split(" ");		
		
		System.out.print("Buscar no texto: ");
		Scanner busca = new Scanner(System.in);
		
		String pesquisa = busca.nextLine().trim();
		
		busca.close();
		
		int contador = 0;
		for (int i = 0; i < fracionado.length; i++) {
			if (pesquisa.equalsIgnoreCase(fracionado[i])) {
				contador++;
			}
		}
		
		if (contador == 1) {
			System.out.println("O termo '" + pesquisa + "' foi encontrado " + contador + " vez.");
		}else if (contador > 1) {
			System.out.println("O termo '" + pesquisa + "' foi encontrado " + contador + " vezes.");
		}else {
			System.out.println("O termo '" + pesquisa + "' não foi encontrado.");
		}
	}
}

Essa solução eu separei em alguns passos:

  • 1º Passo - Determinar a String que será analisada pelo programa e transformá-la em Array.

  • 2º Passo - Abrir um Scanner para que o usuário digite o termo que ele deseja buscar e armazená-lo com o método trim() para remover quaisquer espaços em branco digitados antes ou depois do termo pesquisado.

  • 3º Passo - Iniciar a variável contador igual a 0, e em um laço de repetição que percorrerá todos os endereços armazenados Array, vai ignorar as letras maiúsculas e realizará a iteração de soma do contador.

  • 4º Passo - Por fim, a saída em forma de estrutura condicional onde vai retornar um texto para 1 termo, outro para mais do que 1 e outro para termo não encontrado.

Saída:

Buscar no texto: texto
O termo 'texto' foi encontrado 3 vezes.

  Uma possível melhoria nesse código seria ignorar pontuações coladas nas palavras (vírgula, ponto. "aspas" (parênteses), etc.), pois tive que afastar na String analisada para rodar o programa corretamente. E também ignorar os acentos porque apenas termos digitados com acentuação correta que entram na busca.

Anagramas

  Você deve criar um programa que permita ao usuário digitar duas palavras e fazer a análise se elas são anagramas. Exemplo 'amor' tem as mesmas letras de 'Roma', portanto, são anagramas.

Resolução 
import java.util.Arrays;
import java.util.Scanner;

public class Anagrams {

	public static void main(String[] args) {
		System.out.println("--Verifique se duas palavras são anagramas--");
		
		Scanner sc = new Scanner(System.in);
		
		System.out.print("Primeira palavra: ");
		String entrada1 = sc.nextLine().toLowerCase().trim();
		String[] fracionado1 = entrada1.split("");
				
		System.out.print("Segunda palavra: ");
		String entrada2 = sc.nextLine().toLowerCase().trim();
		String[] fracionado2 = entrada2.split("");
		
		sc.close();
		
		Arrays.sort(fracionado1);
		Arrays.sort(fracionado2);		
		
		if (Arrays.deepEquals(fracionado1, fracionado2)) {			
			System.out.println(entrada1 + " e " + entrada2 + " são anagramas!");			
		} else {			
			System.out.println(entrada1 + " e " + entrada2 + " não são anagramas!");
		}	
	}
}

Passos para a resulução:

  • 1 - Abrir um Scanner para receber as entradas digitadas pelo usuário.

  • 2 - Solicitar ao usuário a digitação da primeira e segunda palavras, separadas. Cada uma delas utilizando os métodos .toLowerCase() para converter todo o conteúdo digitado em letras minúsculas para a comparação, e .trim() para eliminar espaços vazios que possam ser digitados por engano.

  • 3 - Converter o conteúdo da String em um Array com o método .split("") passando o parâmetro ("") sem espaços, para o programa realizar o corte em cada caractere, para análise.

  • 4 - Utilizar o método Arrays.sort() para cada Array a fim de ordenar seus elementos.

  • 5 - Por fim uma estrutura condicional para controlar o fluxo de saída do programa, comparando os dois Arrays e indicando se as palavras são ou não anagramas.

Saída:

--Verifique se duas palavras são anagramas--
Primeira palavra: amor
Segunda palavra: RomA
amor e roma são anagramas!
