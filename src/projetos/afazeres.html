Afazeres - Lista de Tarefas Para Navegador

  Dentro do meu ambiente de desenvolvimento eu sempre tenho alguma tarefa agendada para resolver, um bug no programa x, uma correção do layout no programa y. Como meu ajudante e organizador eu utilizo o WhatsApp numa conversa comigo mesmo e vou enviando cada tópico em uma mensagem e vou deletando conforme finalizo as tarefas.

  Com essa idéia de agilizar o meu processo, desenvolvi essa aplicação de navegador onde é possível adicionar uma tarefa, marcar ela como realizada, mantendo ela riscada na tela, para controle, e com a opção de deletá-la também.

edit 18/01/2025
  Adicionado o armazenamento local no navegador. Funciona apenas na aba normal, não funcionando em aba anônima.
